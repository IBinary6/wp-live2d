== Installation ==
This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-live2d` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Re-Activate it through the "Live2D" menu in WordPress
4. Create a Widget for Live2D
